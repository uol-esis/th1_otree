What you need to know

In this study, your task is to transform datasets into a structure that can be used reliably for analysis. This means you will prepare the data so that it follows the principles of a tidy dataset.

Structuring data in this way is important because it ensures that datasets can be processed, analyzed, and combined without errors. The tidy format provides a clear and consistent structure, which makes it easier to clean, visualize, and analyze data.

i

A tidy data record has to satisfy the following requirements:

  • Each column represents one variable
  • Each row represents one observation
  • Each cell contains exactly one value
